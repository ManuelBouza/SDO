# Scope and Non-goals

## Scope

SDO applies to operational work across Linux, Windows, CLIs, APIs, databases, Kubernetes, cloud, IaC, GitOps, pipelines, manual procedures, and AI-assisted operations.

## Non-goals

SDO does not replace:

- ITIL or ITSM
- SRE
- GitOps
- Infrastructure as Code
- CI/CD
- Runbooks
- Incident response
- Ticketing systems
- Human accountability

## Relationship to other practices

SDO provides a common operational contract that can wrap or reference those systems without duplicating them.
