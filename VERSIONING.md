# Versioning

SDO uses semantic versioning for the methodology.

```text
MAJOR.MINOR.PATCH
```

## Rules

| Change | Version |
|---|---|
| Changes the base lifecycle, required artifacts, or conformance rules | MAJOR |
| Adds compatible profiles, gates, templates, or integrations | MINOR |
| Clarifies wording, examples, or non-breaking guidance | PATCH |

Current draft version: `0.1.0`.
